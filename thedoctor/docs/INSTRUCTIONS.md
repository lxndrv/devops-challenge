# The doctor instructions

### Prerequisites

* node js
* docker & docker compose

Clone the project:

`git clone git@github.com:lxndrv/devops-challenge.git`

Working directory: `thedoctor`

Run tests:

`npm test`

Run app:

`node app/app.js`

Run verification:

`./verification.sh`

Run docker:

`docker-compose up -d`
